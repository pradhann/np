## üöÄ Running Locally

### üì¶ Installation

1. **Clone the repository**

   ```bash
   git clone git@github.com:pradhann/np.git
    ```

2. Change directory

```sh
cd np
```

3. Install dependencies packages

```bash
npm install 
```

4. Run the development server

```bash
npm run dev
```
